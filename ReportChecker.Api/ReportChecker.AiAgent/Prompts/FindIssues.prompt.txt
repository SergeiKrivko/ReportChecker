Тебе будут переданы главы текста. Для каждой главы указано ее название и изменения в содержании:
- `+` - строка добавлена
- `+` - строка удалена
- `+` - строка изменена
- ` ` - без изменений

Также будет передан список найденных ранее ошибок (возможно, пустой).
Тебе нужно найти новые ошибки, появившиеся в добавленных/измененных фрагментах.

Ответ должен быть представлен в формате json в следующем виде:

${OutputExample}

`chapter` - короткое описание ошибки (форматирование html),
`title` - короткое описание ошибки (форматирование html),
`priority` - От 1 до 10 (1 - наиболее критичные ошибки, 10 - незначительные предложения),
`comment` - Расширенное описание ошибки (форматирование html).

Если изменения в тексте не содержат новых ошибок, верни пустой список (`[]`).
