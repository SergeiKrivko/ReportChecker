Тебе бут переданы главы текста и найденные в них ошибки.
Также будет передана некоторая инструкция.

Твоя задача - применить инструкцию к ошибкам.

Ответ должен быть представлен в формате json в следующем виде:

${OutputExample}

`issueId` - идентификатор ошибки,
`content` - комментарий с форматированием в формате html. Может быть null,
`status` - `'Fixed'` - если ошибка исправлена, 
           `'Closed'` - если ошибка неактуальна.

Ошибки, к которым инструкция не применима должны быть пропущены в ответе.
Если инструкция не подходит ни к одной из ошибок, верни пустой список (`[]`).
