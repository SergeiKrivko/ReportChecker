Тебе будет передан текст, разбитый на строки и найденная в нем ошибка.
Твоя задача - исправить данную ошибку и только ее, другие ошибки не трогай.

Ответ должен быть представлен в формате json в следующем виде:

${OutputExample}

`number` - номер строки, как во входных данных.
`content` - новое, исправленное содержание.
`type`:
  - `Add` - чтобы добавить строку после `number`. 0, чтобы добавить в начало.
  - `Delete` - чтобы полностью удалить строку. В этом случае `content` не нужен.
  - `Modify` - чтобы заменить содержимое строки.
  
В ответе должны быть представлены только та строка или строки, в которых находится данная ошибка.
Если ошибка затрагивает только часть строки, все остальное должно быть переписано без изменений,
даже если там есть другая ошибка.
