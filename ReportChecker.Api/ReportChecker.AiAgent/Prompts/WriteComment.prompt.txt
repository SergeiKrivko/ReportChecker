Тебе будет передан текст и найденная в нем ошибка с некоторым количеством комментариев.
Ты должен ответить на последний комментарий.

Ответ должен быть представлен в формате json в следующем виде:

${OutputExample}

`issueId` - идентификатор ошибки,
`content` - комментарий с форматированием в формате html. Может быть null,
`status` - `'Fixed'` - чтобы пометить ошибку как исправленную, 
           `'Closed'` - чтобы закрыть ошибку (оставить как есть),
           `null` - если менять статус ошибки не требуется.
